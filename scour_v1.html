장기세굴 (Long-term Scour)

과거 하상측량 자료의 하상고 변화 추세로 장기세굴심을 산정합니다.
장기세굴심 = 과거 하상고(기준) − 현재 하상고 — 하상이 낮아진 만큼이 세굴심입니다.
측량자료가 없는 경우 HEC-6 모형 또는 직접입력 방식을 선택하세요.

입력 데이터

계산식: ys,lt = 과거 하상고 − 현재 하상고
예시: 과거 EL. 9.50m − 현재 EL. 9.00m = 0.50m 장기세굴
주의: 결과가 음수(하상 상승)이면 0으로 처리합니다
HEC-6 / Blench 공식 등 별도 분석 결과를 직접 입력하세요.
권고: 교량 계획기간(100년) 동안의 하상저하 누적량 적용
📐 적용 지침: 하천설계기준 3.3.3 / FHWA HEC-18 Ch.3 — 과거 최심 하상고(또는 HEC-6 예측 하상고)와 현재 하상고의 차이를 보수적으로 적용합니다.

수축세굴 (Contraction Scour)

교량부에서의 하도 수축에 의한 세굴. 임계유속(Vc) 비교로 Live-bed / Clear-water 조건을 자동 판별하여 HEC-18 공식을 적용합니다.

수리 입력 데이터

이송 모드 (k₁) 선택

Live-bed: y₂ = y₁ × (Q₂/Q₁)^(6/7) × (W₁/W₂)^k₁
Clear-water: y₂ = [Q₂² / (Ku² × Dm^(1/3) × W₂²)]^(3/7)
Ku = 0.025 (SI), Dm = 1.25 × D₅₀ (m)
