# 교량 교각 세굴심 검토 툴

교량 안전점검 시 교각 국부세굴심 산정 및 등급판정 자동계산 툴

## 파일 구성

| 파일 | 버전 | 설명 |
|------|------|------|
| `local_scour_v3.html` | **v3.0 (최신)** | 국부세굴 전용 · 4개 공식 · 표1.62/1.63/1.64/1.65 등급판정 |
| `local_scour_v2.html` | v2.0 | 국부세굴 전용 · 4개 공식 · 기초형식별 단면도 |
| `scour_v1.html` | v1.0 | 장기/수축/국부 3종 통합 툴 |

## v3.0 주요 기능

### 세굴심 산정 (그림 1.35 기준)
- CSU, Neill, Laursen, Froehlich **4개 공식 동시 산정**
- **최댓값 제외 나머지 3개 평균** 적용 (시설물 안전점검 지침 그림 1.35)
- 분수식 수식 실시간 표시

### 등급 판정 (표 1.62 ~ 1.65)
- **[표 1.62]** 사전판정: 우물통기초/암반지반/저항성충분 → 즉시 a등급
- **[표 1.63]** 기본등급: 기초 내 세굴선 침투율로 a~e 자동 판정
- **[표 1.64]** 보호공 보정: b/c/d등급에 상·중·하 보정 적용
- **[표 1.65]** c/d등급 → 2단계 기초안전성 검토 안내

### 단면 개념도
- **직접기초 / 말뚝기초** 형식별 SVG 자동 전환
- EL.0 / EL.1a / EL.1 / EL.2 체계 실시간 반영
- 등급별 칼라 바 및 세굴선 포인터 표시

### 수리 계산
- Froude수, 임계유속 Vc 자동 계산
- Live-bed / Clear-water 조건 자동 판별
- 투영교각폭 b' 자동 계산 (b·cosθ + L·sinθ)

## 적용 기준

- 시설물 안전점검 지침 (국토교통부) 표 1.62~1.65, 그림 1.35
- FHWA HEC-18 (Evaluating Scour at Bridges, 5th Ed.)
- KDS 51 90 10 하천교량 (2018)
- 도로교설계기준 (한계상태설계법)
